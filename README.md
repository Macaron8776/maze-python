# 【２人対戦型　迷路脱出ゲーム】

大学生の研究室で作成したアプリです。
２人で対戦できる、ライフ・アイテム制を導入した、迷路脱出型ゲームです。

## 💡 開発背景と目的

* 大学の研究室では、Pythonを用いたプログラミングで何かアプリを製作することでした。そのため、ゲーム制作にチャレンジをしたく、作成いたしました。

---

## 🚀 主な機能

* ** ２人対戦形式 **：２人で対戦する形式です。１人目のプレイヤーは[矢印キー]で、２人目のプレイヤーは[SZXC]で操作します。
* ** アイテムとスコア制の導入 **：アイテムを取得することでスコアがアップする機能を導入しました。
* ** 敵とライフ制の導入 **：動く敵とライフ制を導入し、ライフが０になるとゲームオーバーになる機能を導入しました。
